## 🚀 Déployer sur GitHub Pages

### Étape 1 : Créer un repo GitHub

1. Allez sur https://github.com/new
2. Nommez le repo : **`code-du-succes`**
3. Cochez **"Add a README file"**
4. Cliquez **"Create repository"**

### Étape 2 : Uploader les fichiers

1. Dans votre repo, cliquez sur **"Add file"** → **"Upload files"**
2. Glissez-déposez les fichiers du dossier `workspace/code-du-succes/` :
   - `index.html`
   - `.gitignore`
   - `README.md`
3. Cliquez **"Commit changes"**

### Étape 3 : Activer GitHub Pages

1. Allez dans **Settings** (Paramètres)
2. Cliquez sur **"Pages"** (dans le menu de gauche)
3. Sélectionnez **"Deploy from a branch"**
4. Choisissez la branche : **`main`**
5. Choisissez le dossier : **`/(root)`**
6. Cliquez **"Save"**

### Étape 4 : Accédez à votre site 🎉

Votre page sera disponible à :
```
https://[votre-username].github.io/code-du-succes/
```

GitHub mettra 1-5 minutes à déployer la page.

## 🔧 Personnalisation

Pour modifier la page, éditez `index.html` :

- **Couleurs** : Changez les codes couleur (recherchez `#667eea`, `#764ba2`)
- **Texte** : Modifiez directement dans les sections HTML
- **Prix** : Changez `€17.99` et `€30`
- **Témoignages** : Remplacez les noms et textes
- **Lien Beacons** : Modifiez l'URL dans la section `<script>`
